关于这篇博客的搭建

在搭建好了一个博客后，通常的做法就是写一篇心得体会，总结一下如何搭建这样的博客。

免不了俗套，我也简单记录一下利用 GitHub Pages 搭建基于 Hexo 的博客的经验，好让自己或者后人借鉴。

搭建流程

在 GitHub 上创建一个账号

不知道网址的直接点这个链接GitHub

在 GitHub 上新建一个账号同名Repo

Repo名为username.github.io，Repo 一般设为 Public，因为在 GitHub 上的私有仓库是收费的。

添加 SSH Key

在 GitHub 账号的 Settings 中，会有SSH and GPG Keys的选项，这里就是待会要填写信息的地方。在 Mac 上打开 Terminal，进入用户目录，输入ssh-keygen -t rsa -C "your_email@example.com"，会在.ssh文件夹下产生id_rsa和id_rsa.pub，分别是你的私钥和公钥。请将id_rsa.pub的内容粘贴到 GitHub 上并保存。

安装 Node.js

由于开发的工具是 Mac，于是我直接去NodeJS官网下载，一般下载 LTS 版本。如果你比较激进，也可以选择最新版，但可能会有一些 bug，不过谁知道呢。

安装 Git

只要你在 Mac 上安装了 XCode 的开发者工具，Git 就应该已经安装好了。可以通过命令git --version在 Terminal 中查看。
通常安装好 Git 后，在第一次使用 Git 前需要设置你的用户名和邮箱。命令如下：

接着，使用git config --list就可以查看到你刚刚设置的用户名和邮箱了。

安装 Hexo

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。
在安装 Hexo 前，请确认安装了如下软件：

  • Node.js
  • Git

然后在 Terminal 中输入npm install -g hexo-cli即可进行安装。

创建 Hexo 目录

打开 Terminal 输入hexo init username.github.io，即可在本地创建你的博客目录。

安装NexT主题

NexT 主题的网址。网站上对使用 NexT 主题进行了详细的介绍。

配置主题

配置主题在 NexT 的网站上已经有相应的介绍，你可以根据自己的喜好进行配置。

部署 Hexo

部署 Hexo 需要注意的一点是需要利用npm install hexo-deployer-git --save提前安装hexo-deployer-git自动部署工具，否则会报错。
接着，利用hexo clean && hexo g && hexo d命令即可将其部署到 GitHub Pages 上。
