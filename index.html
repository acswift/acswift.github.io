关于 Ruby

  1. Ruby 是脚本语言
  2. Ruby 是面向对象的语言
    • Ruby 具有继承、Mix-in 等面向对象语言必备的特性。
    • Ruby 不仅提供了丰富的标准类库，还有对应各种异常的错误处理机制、自动释放内存的垃圾回收机制等提高编程效率的特性。
  3. Ruby 是跨平台的语言
    • Ruby 能在 Mac OS X、Linux、FreeBSD、Solaris 等类 Unix 操作系统以及 Windows 操作系统等平台上运行。它的大部分脚本无需修改即可在各个不同的平台环境下运行。
  4. Ruby 是开源软件

  • 基因：属性 和 方法（变量和函数）
  • 繁殖：完全克隆 和 基因交换（继承和变异）
  • 术语：结构体（Struct） 和 类（Class），Swift 3 开始推荐自己定义的对象使用结构体。
