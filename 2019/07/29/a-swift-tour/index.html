A Swift Tour Part 01

发表于 2019-07-29 分类于 编程

Swift 是一门为 iOS、OS X 以及 watchOS 应用开发而开发的构建于 C 和 Objective-C 的优点之上的新编程语言。

关于 Swift

  • 它没有 C 的兼容性限制。
  • Swift 语言由成熟且倍受喜爱的 Cocoa 和 Cocoa Touch 框架支持，设想一下，这将是一个重新定义软件开发的契机。
  • 我们使用自动引用计数（ARC）简化了内存管理，我们的框架栈牢牢地打造在 Foundation 和 Cocoa 框架之上，使现代化和标准化贯穿始终。
  • Objective-C 语言本身已经支持块、集合以及模块，无中断引入现代语言技术。
  • Swift 语言引入了 Objective-C 命名参数的可读性以及其动态对象模型的能力。它提供了无缝访问现存 Cocoa 框架以及与 OC 混合编写的兼容性。

版本兼容性

当你使用 Xcode 9 来编译 Swift 3 代码，Swift 4 中的大部分新功能都是可用的。只有如下功能是 Swift 4 代码独享的：

  • 拆分字符串操作返回 Substring 类型，而不是 String ；
  • @objc 标签在更少的地方被推断；
  • 同一文件中类型的扩展可以访问类型中的私有成员。

用 Swift 4 编写的编译目标可以依赖于用 Swift 3 编写的编译目标，反之亦然。也就是说如果你的项目很大且分成了多个框架，你可以一次从 Swift 3 迁移到 Swift 4 一个框架。

Hello, world!

依照传统，使用新语言写的第一个程序都应该是在屏幕上打印 “Hello,world!” ，使用 Swift 语言，你可以在一行中完成。

常量

预期范围内恒定不变的量

常量的定义

  • let 常量名
  • 常量名可以是任意文字，长度不限

常量赋值

  • 给常量一个值，术语叫“赋值”
  • 形式：let 常量名 = 值

变量

可主动变更或者可预计期间内自动有规律或者无规律的量，比如清醒时间、风扇转动的总次数、加班时间、股票指数、驾照通过的几率。。。

代码注释

  • 有助于自己理解
  • 便于他人理解
  • 专业优雅

如何正确使用常量与变量？

  • 总是优先使用常量
  • 交给编译器决定

值的类型和类型推断

  • 3 这样的整数，在 Swift 中称为 Integer 类型的值，简写 Int
  • 形式：var 变量名: Int = 3. 由于 Swift 有类型推断，类型可以省略不写。

浮点型（小数）

  • Swift 中默认浮点型是 Double(双精度)

类型安全

  • 变量一旦定义，其类型不可更改，即：不能给变量一个类型不同的值。

布尔型：Bool, 表示逻辑上的真或假

  • 只有 2 个值，true 和 false.

元组（Tuple）：定义变量的一个组合

  • 形式（一般）：(3, “day”, “Swift”, “3.0”)
  • 形式（前缀）：(day: 3, unit: “day”, lang: “Swift”, ver: “3.0”)

可选类型（Optional）：代表变量可能有值的情况

  • 如用户资料的选填部分，如住址/性别等辅助信息
  • 形式：var 变量: 类型? , 默认是无值（nil）

基础操作符

略

字符串和字符

字符串

  • 字符串是有序的字符集合，或者叫文本，如“我已使用量洪荒之力”。
  • String 是字符串类型，Character 是字符类型。
  • 2 个字符串可以通过 + 来连接。
  • 通过字符串插值可合成一个长字符串。
  • 空字符串用 "" 表示。
  • 判断字符串是否为空

字符

  • 用 Character 类型来定义字符串。
  • 可以对一个字符串的 characters 属性进行循环，来访问其中单个字符。

连接字符串和字符

  • 用 +。
  • 向 字符串 添加 字符，用 append 方法。

字符串插值：组合常量/变量/字面量/表达式成一个长字符串

集合类型

一组同类型的值的集合，根据组合的整体特性分为：

  • 有序可重复 - 数组（Array）
  • 无序不重复 - Set
  • 无序可重复，但每个值有唯一的键（Key）- 字典（Dictionary）
  • 批量处理集合中的元素，可以使用 for-in 循环

数组

  • 有序可重复，用于存储同一类型的值
  • 定义：Array <类型> 或 [类型]
  • 创建一个有默认值的数组
  • 创建一个有序范围的[Int]数组，Array(起始值…终止值)
  • 用数组字面量来创建数组：[值1，值2，值3，…，值N]
  • 元素计数：count，空否：isEmpty
  • 添加：append，添加一个同类型数组 +=
  • 获取元素：数组名[索引]，索引总是从 0 开始
  • 插入：insert
  • 移除：remove

集合

  • 值无序不重复。适合存储具有唯一特性的数据，如用户名/身份证号码/护照号等

  • 定义：Set <元素类型>，无法使用类型推断，但可省略类型

  • 用数组字面量创建集合
  • 元素计数：count, 空否：isEmpty
  • 插入：insert
  • 移除：remove
  • 是否包含某元素：contains
  • 转换为数组：sorted
  • 集合间的运算：交差并补

交集 intersection

差集 subtract

并集 uninon

补集 symmetricDifference

  • 相等：2 个集合有完全相同的元素
  • 子集：isSubset（可以相等），严格子集 isStrictSubset
  • 父集：isSuperset（可以相等），严格父集 isStrictSuperset
  • 无交集：isDisjoint

字典

  • 值无序可重复，但每个值有唯一的键（Key）
  • 定义：Dictionary<键类型, 值类型>，或[键类型: 值类型]
  • 用字典字面量来创建字典，键与值之间用冒号分开
  • 计数：count，空否：isEmpty
  • 添加或更新：字典变量[键] = 值
  • 获取，依然可以用下标
  • 移除：用下标把值设为 nil
  • 循环一个字典 for-in，因为键值对有 2 个元素，用远组变量
  • 单独使用其中键或值，使用 Keys 或 Values （可使用 for-in）
  • 把键值对分离成数组，用 [数组类型](字典变量.keys)，[数组类型](字典变量.values)

控制流

  • 用来检查和运算各种条件逻辑和逻辑的组合，控制 APP 的运行流向
  • 循环语句 for-in, while
  • if 和循环控制语句
  • 早退语句 guard else
  • 退出循环语句 break, continue
  • switch

for-in 循环

  • 执行固定次数的操作，或操作一个序列（如数组）中的每一项
  • 序列中的每项都参与。for 与 in 之间加一个代替每项的名
  • 只需循环固定次数，可以省略项名。比如打印 5 次相同的文本

while 循环

  • 执行一系列操作，直到条件不成立。使用场合：执行次数未知
  • 格式：while 条件 {语句}。比如计算 1 到 100 的和。

循环中的判断

  • continue 结束本次循环，break 结束整个循环

switch

  • switch 语句：超过 2 个以上条件判断的场合
  • 条件匹配模式
  • 范围匹配：如根据室外温度来自动调节室内温度的空调
  • 元组匹配：如某坐标（含 x 和 y 2 个变量）所在象限判断

函数

  • 函数是一段代码块和一个会意的名字。用于执行特定的任务
  • 使用函数叫“调用（call）”
  • 就像公式一样，使用函数需要提供参数
  • 函数调用的结果叫返回值
  • 函数都有类型，包含参数和返回值类型

定义函数

  • 形式：func 函数名(参数1:类型, 参数2:类型, …) -> 返回结果的类型 {执行语句}
  • 调用：var 变量名称 = 函数名(变量1, 变量2, …)
  • 🌰：加法函数

参数和返回值：可以无参数无返回值

  • 无参数无返回值。一般用于执行一系列操作，不需要结果。
  • 多返回值（使用元组）
  • 参数以 名称: 类型 列出，多个参数间用逗号分隔
  • 可以给某个参数以默认值
  • 函数类型：包含参数和返回类型的简写形式，可以像普通变量那样使用，一般用于函数式编程。🌰(Int, Int) -> Int

闭包 Closure

  • 闭包是一种函数的简写形式，省去函数名，把参数和返回值放入花括号内

闭包表达式

  • 有时候需要更简洁的使用函数的方式
  • 尤其是一个函数的某个参数是另一个函数时
  • 这节主要以 sorted 函数为例
  • sorted 函数用于对数组排序，只接受一个函数型参数，描述排序逻辑

闭包的自动推断

  • 参数和返回类型可以自动推断，单表达式可以忽略 return 关键词
  • 可使用快捷参数，前缀 $ 从 0 开始递增

枚举 enumeration

  • 计数、列举（有限的情况）；详细叙述之意。“增强的、更抽象型”的数字类型
  • 非常强大，与 switch 语句结合使用，天生一对

定义和使用

  • 给定一个名称，然后把每一种情况列举出来
  • 使用：点语法
  • 与 switch 配合使用。如果变量是枚举值，可省略枚举名

附加值

  • 赋予附加值：加上元组字面量
  • 取得附加值：用 switch 语句，在 case 中加上元组变量
